# 동적 가상 클래스

<br/>

## 하이퍼링크 가상 클래스

### :link

- href 속성이 존재하고, 방문하지 않은 주소를 가르킨다.

### :visited

- 이미 방문한 주소를 가르킨다.
- 보안상의 문제로 적용할 수 있는 스타일이 제한적이다.

<br/>

## 사용자 행동 가상 클래스

### :focus

- 현재 입력 포커스가 있는 요소

### :hover

- 현재 마우스 커서가 올라가 있는 요소

### :active

- 사용자 입력에 의해 활성화된 요소
- a태그의 경우, 클릭중인 상태를 의미한다.
